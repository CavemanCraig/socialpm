What should I call it?

The project name should be something that you and all of your partners will easily recognize and associate with. Try not to use things like dates and general topics.
Good examples:
  • Acme Website Rewrite
  • ACME-252
Bad examples:
  • 2011-08-01
  • Website Rewrite

What about the vision?

Vision statements should be both specific and measurable. Try to be as clear and concise as possible.
Good examples:
  • Less expensive than our top five competitors.
  • No page more than three clicks away.
Bad examples:
  • Inexpensive.
  • Easy to access.
